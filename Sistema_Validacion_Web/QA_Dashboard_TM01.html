Bienvenido al Sistema de Validación QA

Como Quality Assurance, tu rol es validar, revisar y aprobar las especificaciones técnicas de los sistemas ITS del proyecto TM01. Tienes acceso a todas las funciones de validación y control de calidad.

📊 Validar WBS y Presupuesto

Revisar la estructura de trabajo (WBS) y validar el presupuesto de los sistemas ITS. Verificar cantidades, costos y coherencia técnica.

📋 Validar Cumplimiento Contractual

Verificar el cumplimiento de las especificaciones técnicas vs las obligaciones contractuales. Identificar discrepancias y generar DT.

📈 Generar Reportes de Validación

Crear reportes automáticos para Interventoría y ANI. Exportar documentación de validación y seguimiento.

📝 Revisar Documentos Técnicos (DT)

Revisar, aprobar o rechazar los Documentos Técnicos generados por los Especificadores. Control de calidad documental.

🔧 Validar Sistemas ITS Específicos

Validar cada sistema ITS individualmente: SOS, ETD/RADAR, CCTV, PMV, Meteo, WIM, Fibra Óptica y CCO.

📊 Dashboard de Validación

Acceder al dashboard principal de validación con estado general de todos los documentos y sistemas del proyecto.
