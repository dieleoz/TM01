# SOLICITUD DE CAMBIO (DT-001) - SISTEMA SOLAR CCTV

**Fecha:** 2026-01-23  
**Sistema:** CCTV  
**Item:** 3.1.14  

## 1. Antecedentes
El ítem original "Kit Energía Solar 100W" (Cant: 8) provenía de una ingeniería conceptual antigua para nodos remotos específicos. No cubría la totalidad de las cámaras ni cumplía con la autonomía requerida para los equipos PTZ de alto rendimiento definidos.

## 2. Descripción del Cambio
Se ajusta el ítem para cubrir la alimentación de las 15 cámaras CCTV mediante solución autónoma solar.

### Especificaciones Nuevas
- **Panel Solar:** 540W Monocristalino (Mayores dimensiones y eficiencia).
- **Almacenamiento:** Batería GEL 200Ah @ 12V (Ciclo profundo).
- **Control:** Controlador MPPT 30A/40A Industrial con comunicación.
- **Infraestructura:** Gabinete IP65 + Soportes de poste reforzados.
- **Alcance:** 1 kit por cada punto de cámara (15 Unidades).

El cambio es **técnicamente necesario** para la operatividad del sistema y **financieramente favorable**, generando un ahorro del 53% en este rubro a pesar de duplicar la cantidad de unidades.
