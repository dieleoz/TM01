# DT-TM01-002: Eliminación Sistema Radio FM

## 1. Información General
- **ID Cambio:** DT-TM01-002
- **Fecha:** 22/01/2026
- **Solicitante:** Gerencia de Proyecto / Cliente
- **Sistema Afectado:** Subsistema 12 - Radio FM
- **Estado:** Aprobado para Implementación

## 2. Descripción del Cambio
Se elimina del alcance el **Subsistema de Radio FM** (Estación Base + Repetidoras).
- **Alcance Anterior:** 1 Estación + 5 Repetidoras.
- **Alcance Nuevo:** 0 Unidades.

## 3. Justificación TÃ©cnica y Contractual
1.  **Revisión Contractual:** El Apéndice Técnico 1 (AT1) y AT3 no estipulan la obligatoriedad de una emisora propia del proyecto. La comunicación con el usuario se suple mediante:
    *   Paneles de Mensajería Variable (PMV).
    *   Postes SOS.
    *   Redes Sociales / Apps (Gestión CCO).
2.  **Viabilidad Económica:** El costo de $880 Millones COP (~$231k USD) no representa un valor agregado proporcional al beneficio, considerando la cobertura de redes celulares 4G/5G en el corredor.
3.  **Complejidad Operativa:** Se eliminan los trámites de licencias de espectro ante MinTIC y ANE.
