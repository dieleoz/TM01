4. PLAN DE INTEGRACIÓN DE DATOS (T05)


La "Magia" del Smart CAPEX reside en cómo se presentan los datos al SCADA para cumplir el contrato sin llenar la vía de postes.


  • Volumen de Tráfico: Se inyecta el dato de los Peajes (Zambito/Aguas Negras) y se aplica un factor de corrección por UF en el software.
  • Velocidad Promedio: Se consulta API de Tráfico (Google Maps Platform) cada 5 minutos para "pintar" la velocidad del tramo en el mapa.
  • Detección Incidentes: Se usan los 2 ETD Físicos + Cámaras CCTV PTZ (Presets) para alarmas visuales.
