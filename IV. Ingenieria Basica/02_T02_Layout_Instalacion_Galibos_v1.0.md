## 2. CONFIGURACIÓN DIBUJO TÍPICO

### 2.1 Componentes
*   **Detector:** Sensor Láser/Infrarrojo de barrera o escáner.
*   **Aviso:** Señal luminosa LED "ALTO - GÁLIBO EXCEDIDO" (Activada por sensor).
*   **Estructura:** Poste dedicado o montaje sobre estructura existente (Puente).

### 2.2 Lógica de Operación
1.  Vehículo excede altura permitida.
2.  Sensor detecta intrusión.
3.  Se activa sirena y baliza visual.
4.  Cámara CCTV (asociada) toma foto de evidencia.
